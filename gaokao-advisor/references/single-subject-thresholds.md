# 单科成绩硬门槛速查

> 部分高校在招生章程中对语文/数学/外语/选考科目的单科成绩有明确要求。学生总分够、但单科不够，照样退档。

## 一、外语类专业（最常见）

### 顶尖外语类院校
| 院校 | 英语单科要求 |
|------|-------------|
| 北京外国语大学 | 通常 ≥ 130（满分 150） |
| 上海外国语大学 | 通常 ≥ 130 |
| 北京第二外国语学院 | 通常 ≥ 120 |
| 广东外语外贸大学 | 通常 ≥ 120 |
| 大连外国语大学 | 通常 ≥ 115-120 |

### 综合类院校的英语类专业
- 985/211 综合大学的"英语"、"翻译"、"商务英语"专业：通常要求 ≥ 120 或 ≥ 125
- 部分校的"英语口语"专业要求高考英语口语成绩

### 关键提示
- 提前批小语种（俄语、日语、阿语、西语等）通常英语 ≥ 110-120
- "外国语言文学类"大类招生中，分流到英语方向时再次审视英语成绩

## 二、数学类专业

### 顶尖数学院校
| 院校 | 数学单科要求 |
|------|-------------|
| 北京大学数学学院 | 通常 ≥ 130（强基） |
| 清华大学数学系 | 通常 ≥ 130（强基） |
| 中科大数学（少年班、英才班） | ≥ 135 |

### 211 / 985 综合大学的"数学与应用数学"
- 部分校要求数学 ≥ 120 或 ≥ 125
- "金融数学"、"信息与计算科学"专业近年也有数学要求

### 关键提示
- 经济学、金融工程、统计学、计算机大类招生在分流时通常考察数学成绩
- 数学 < 100 的学生不建议报 CS / 金融工程 / 数据科学

## 三、语文类专业

### 师范类
- 汉语言文学（师范）：部分校要求语文 ≥ 110
- 部属师范的中文类专业：≥ 115 较常见

### 新闻传播类
- 中传、中国人民大学新闻：语文 ≥ 115（部分校）

## 四、医学类（综合考量，单科要求较少）

医学类多数不设单科硬门槛，但：
- 部分校要求**英语 ≥ 120**（医学文献需双语阅读能力）
- 临床医学（八年制 / 5+3）多数有英语要求
- 部分校要求**理综/化学单科 ≥ X**

## 五、新高考再选科目要求（隐性单科）

新高考省份的"再选科目"赋分后的要求：
- 部分顶尖工科要求**化学赋分 ≥ 80**（赋分制下）
- 部分医学要求**生物赋分 ≥ 75**

## 六、本流程 处理流程

### 信息采集阶段
反问学生总分时，**同时要求列出语数外三科分数**（再选科目可选）。

### 候选筛除阶段
对每所候选院校，**搜索该校招生章程的单科要求**：
```
[高校名] 2025 招生章程 单科 英语 数学 要求
[高校名] 2025 招生简章 体检 单科限制
```

### 报告呈现
每所院校的论证段必须标注：
- 该校该专业组单科要求（已查到的）
- 学生单科是否达标
- 不达标时直接从清单移除并说明原因

## 七、找不到单科要求时的处理

如某校招生章程未公开提及单科要求：
- 默认按"无硬性单科门槛"处理
- 但论证段提示用户"建议自行查阅当年招生章程确认"

## 八、近年观察到的退档案例（警示）

- 某 985 法学专业要求英语 ≥ 110，学生英语 105，因总分够但单科不够被退档，落入征集志愿
- 某顶尖外语院校英语 ≥ 130，学生总分超 20 分但英语 128，退档
- 某医学院临床医学要求**化学 + 生物双合格**（赋分 ≥ 70），学生化学 68 被退档

**这是为什么必须采集语数外单科分**。

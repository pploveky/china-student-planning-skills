# 高考体检受限专业速查

> 依据《普通高等学校招生体检工作指导意见》（教育部 2003 年印发，至今为现行通用基础）。具体年度执行细则请以当年招生章程为准。

## 一、依据原则

体检结论分为四类：
- **合格**：可以报考
- **合格但不宜就读**：不录取，但不影响其他专业
- **不予录取**：禁止报考该类专业
- **不能录取**：彻底禁止

学生填报时，**单科限制 + 体检限制叠加判定**，任一不达标即被退档。

## 二、视力相关限制

### 完全禁止类（不予录取）
- **任何一眼矫正视力低于 4.8（小数 0.6）**：飞行技术、航海技术、海洋船舶驾驶
- **裸眼视力低于 4.7**：航海技术、轮机工程（部分院校放宽）

### 限制报考类
| 视力情况 | 受限专业 |
|---------|---------|
| 任何一眼矫正视力低于 4.8 | 海洋技术、刑事科学技术、消防工程、应用气象学、医学影像学（部分校）、口腔医学（部分校） |
| 屈光不正 ≥ 800 度 | 飞行、航海类完全禁止 |

## 三、色觉相关限制（**最常见的红线**）

### 色盲（不能识别红绿黄蓝紫）
**禁止报考**：
- 医学类全部（临床、口腔、护理、医学技术、预防、药学、中医学）
- 化学类全部（化学、应用化学、化工、材料化学）
- 生物类全部（生物科学、生物技术、生物工程）
- 心理学（部分校）
- 食品科学与工程
- 环境工程、环境科学
- 地质学、地球化学
- 油气、采矿、安全工程
- 美术、设计、动画、艺术（部分校）

### 色弱
**禁止报考**：上述大部分（医学、化学、生物核心专业），但**部分校放宽**：
- 临床医学：色弱通常仍禁止
- 心理学、生物工程：部分校接受色弱
- 工业设计、视觉传达：色弱多数禁止

### 关键提示
**色弱学生坚决不要报临床医学**，即使录取也无法毕业（实习中识别血液、试剂、皮肤色变都需要正常色觉）。

## 四、身高相关限制

| 类型 | 男生要求 | 女生要求 |
|------|---------|---------|
| 飞行技术（民航） | ≥ 168cm | ≥ 165cm |
| 空中乘务 / 空保 | ≥ 174cm | ≥ 163cm |
| 军校（部分） | ≥ 162cm | ≥ 158cm |
| 公安联考类 | ≥ 170cm（部分校） | ≥ 160cm（部分校） |
| 海洋技术 / 航海 | ≥ 165cm | 部分校不招女 |
| 旅游管理（部分校） | ≥ 170cm | ≥ 160cm |
| 师范类（部分省专业） | ≥ 160cm | ≥ 150cm（少数省，最近多取消） |

## 五、肝功 / 乙肝 / 其他常见限制

- **乙肝表面抗原阳性**：基本不再作为限制（2010 年后取消歧视性条款），但少数医学类、食品类专业仍有要求
- **谷丙转氨酶高**：医学类、食品类受限
- **结核病**：未治愈不予录取（医学类等）
- **精神类疾病史**：师范、医学、公安类受限

## 六、身体状况其他限制

| 情况 | 受限 |
|------|------|
| 嗅觉迟钝 | 化工、食品、烹饪、消防 |
| 平衡功能障碍 | 飞行、航海 |
| 听力低于 3 米 | 师范、播音、公安 |
| 口齿不清 | 师范、播音、外语 |

## 七、性别隐性限制

虽然《普通高等学校招生体检工作指导意见》要求"无性别歧视"，但部分专业在**招生计划**中明确"仅招男/女"：
- 军校：男女比通常 9:1，部分专业仅招男
- 公安联考：男女分别报录，女生比例小
- 部分采矿、油气、地质、海洋：通常仅招男（实际操作）
- 学前教育、护理：通常优先女生（但近年男护士、男幼师政策性增加）

## 八、本流程 反问话术

第一轮采集时统一询问：

```
"高考体检报告中是否有以下任一受限项：
 - 色盲、色弱
 - 任何一眼矫正视力低于 0.6
 - 任何一眼裸眼视力低于 4.7
 - 身高不足 158 cm（女生）/ 162 cm（男生）
 - 肝功能异常、乙肝表面抗原阳性
 - 其他体检意见中的'不宜就读'/'不予录取'条款？"
```

如学生有任一受限项，必须在阶段 2（候选初筛）就**剔除**对应受限专业，避免投档退档。

## 九、与正式填报的衔接

报告中每所"冲档"院校的论证段，必须额外标注：
- "本组无色觉限制"或"该专业组要求双眼矫正 ≥ 4.8"
- "学生体检与该专业组要求一致 / 存在冲突，建议从清单移除"

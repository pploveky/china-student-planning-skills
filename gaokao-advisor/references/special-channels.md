# 特殊招生通道速查

> 本流程 对特殊招生采用"识别 + 提示"轻量化策略，不深度生成方案。本文件用于辅助识别学生当前所处通道，避免普通批策略冲突。

## 一、强基计划

### 招生学校
39 所原 985 高校（清北复交浙国科等），共 39 所。

### 招生专业
基础学科为主：数学 / 物理 / 化学 / 生物 / 力学 / 历史 / 哲学 / 古文字学 / 中国语言文学 / 基础医学 等。

### 录取流程
1. 高考前报名（一般 4 月）
2. 高考成绩出来后参加校测（笔试 + 面试）
3. 综合成绩 = 高考 85% + 校测 15%（部分校 70/30）
4. **录取在普通批之前**

### 关键节点对话术

```
"是否报名了强基计划？"
  → 未报名：本通道不影响普通批
  → 已报名未入围：本通道不影响普通批
  → 已入围未确认录取：通常 7 月底前确认；若被录取，普通批作废
  → 已确认录取：不需做普通批方案
```

### 与普通批关系
- **强基录取 = 不再参加普通批**（一录取即锁定）
- 强基放弃 = 失去强基录取资格但**不影响**普通批
- 因此强基入围但还未确认时，必须**继续做普通批方案作为备选**

### 适合人群
真心愿意读基础学科（数学/物理/化学等），愿意接受"本博贯通"长周期培养，且有志于学术或基础研究方向的学生。

## 二、综合评价

### 招生省份（截至 2025）
浙江、江苏、广东、山东、上海、湖南（部分） 等。

### 主要类型
- **三位一体**：浙江、上海部分校
- **631 模式**：高考 60% + 校测 30% + 学考 10%（典型如南方科技大学、上海科技大学）
- **511 模式**：少数试点

### 招生学校（部分代表）
南方科技大学、上海科技大学、香港中文大学（深圳）、北京外国语大学（北外）、深圳北理莫斯科大学、中国科学院大学（综评招生）。

### 与普通批关系
- 综评录取 → 不再参加普通批
- 综评未录 → 正常普通批
- 综评校测时间通常在高考后、出分后

### 关键提示
南方科技大学、上海科技大学是综评招生的主力，分数门槛较高（高考成绩通常需达 985 中上水平）。

## 三、提前批

### 主要类型
| 类型 | 代表院校 | 关键特点 |
|------|---------|---------|
| 军校 | 国防科大、陆军工程大学、海军工程大学等 | 政审 + 军检；男女比有限 |
| 警校 | 中国人民公安大学、各省警察学院 | 政审 + 体能测试；男女比例有别 |
| 公费师范 | 6 所教育部直属师范（北师/华师/华中师/陕师/东北师/西南大）+ 部分省属 | 协议服务期 6 年，违约赔偿 |
| 公费医学（农村订单定向） | 各省医学院 | 协议服务乡镇卫生院 6 年 |
| 小语种 | 北外、上外、北二外、中传等 | 部分校仅在提前批招小语种 |
| 航海类 | 大连海事、上海海事、武汉理工航运学院 | 体检受限较多（视力、辨色） |
| 港澳高校（部分） | 香港中文大学、香港城市大学 | 走提前批通道 |
| 公安联考相关 | 部分公安院校 | 毕业后参加联考入警 |

### 与普通批关系
- 提前批录取 → 不参加普通批，但提前批本身**不影响**普通批正常填报
- 提前批未录 → 正常普通批
- **提前批可以"冲"**，不录不退档，给学生一次额外机会

### 关键警示
- 公费师范/医学有 **6 年服务期**，违约赔偿（学费 × 倍数 + 违约金），决定前必须充分了解
- 军校警校有政审 + 体检 + 体能测试三道关，任一不达标即落选

## 四、专项计划

### 三类专项
| 类型 | 招生范围 | 报考资格 |
|------|---------|---------|
| 国家专项 | 教育部直属高校 + 985/211 | 832 个原集中连片特困县学生（已脱贫但维持名额） |
| 地方专项 | 省属重点高校 | 本省指定农村县（具体看各省名单） |
| 高校专项（农村专项） | 95 所部委高校 | 农村户籍 + 学籍 + 三年连续就读 |

### 关键提示
- 户籍必须是**农业户口**（部分省 2014 后取消农业/非农区分，按"农村区域户籍"判定）
- 三个专项可同时报名（不冲突）
- 专项录取后通常**不允许转专业转校**

### 反问话术
```
"户籍是否为农村或贫困县？是否考虑申报国家专项/地方专项/高校专项？"
```

## 五、港澳与中外合办独立招生

### 走独立招生（不占用内地高考志愿名额）
| 学校 | 特点 |
|------|------|
| 香港大学（HKU） | 独立招生，6 月面试 |
| 香港中文大学 | 部分专业走提前批，部分独立招生 |
| 香港科技大学 | 独立招生 |
| 上海纽约大学 | 综合评价招生 |
| 昆山杜克大学 | 综合评价招生 |
| 西交利物浦大学 | 部分综评，部分高考统招 |
| 宁波诺丁汉大学 | 同上 |
| 香港城市大学 | 提前批 |

### 走内地统招（占普通批志愿）
香港城市大学（部分专业）、香港浸会大学、岭南大学（部分）。

### 关键提示
独立招生通常需要**独立报名（约 4-6 月）+ 独立面试（多英文）+ 独立学费（10-15w/年）**，与普通批是平行通道。

## 六、本流程 的处理边界

| 学生情境 | 本流程 行为 |
|---------|------------|
| 已通过强基/综评/提前批录取 | 不为该学生做普通批方案，告知"已录取，不需志愿建议" |
| 报名了特殊通道但未录取 | 正常做普通批方案，作为强基/综评未录情况下的兜底 |
| 想了解强基/综评深度策略 | **超出本流程 范围**，建议学生咨询目标校招办或专门服务 |
| 想了解提前批军警院校 | **超出本流程 范围**，建议学生咨询当地武装部/公安招生处 |
| 普通批 + 同时报名提前批 | 可在普通批方案中提示"提前批是额外机会，不影响普通批，请按需报" |

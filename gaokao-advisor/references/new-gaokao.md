# 新高考省份与选科政策

## 一、各省份高考改革进度（截至 2025 年）

| 模式 | 省份 |
|------|------|
| 3+3（6 选 3） | 北京、上海、天津、海南、山东 |
| 3+3（7 选 3） | 浙江 |
| 3+1+2 | 河北、辽宁、江苏、福建、湖北、湖南、广东、重庆 |
| 3+1+2（第四批） | 安徽、江西、贵州、广西、黑龙江、吉林、甘肃 |
| 3+1+2（第五批） | 山西、内蒙古、河南、四川、云南、陕西、青海、宁夏、新疆、西藏 |
| 传统文理（少数） | 视当年改革进度，需按当年最新公告核实 |

**注意**：分批改革进度每年都有变化，必须在检索时确认当年实际执行的方案。

## 二、3+1+2 模式选科与专业关系

| 选科组合 | 可报专业覆盖率 | 适合人群 |
|---------|--------------|---------|
| 物理 + 化学 + 生物 | ≈ 96% | 想读医、工科、理科、计算机的首选 |
| 物理 + 化学 + 地理 | ≈ 95% | 工科 + 地理科学方向 |
| 物理 + 化学 + 政治 | ≈ 96% | 想报军警、警校、马院、强基的首选 |
| 物理 + 生物 + 地理 | ≈ 75% | 缺化学，工科与医学受限 |
| 历史 + 政治 + 地理 | ≈ 50% | 纯文科组合，专业面窄但传统 |
| 历史 + 政治 + 生物 | ≈ 55% | 偏文 + 部分医学辅助 |

**核心警示**：选化学几乎是工科与医学的入场券。

## 三、3+3 选科要点（以浙江、北京为例）

- 没有"物理 vs 历史"的二选一限制，自由度更高
- 但同样存在专业选考要求：例如计算机大类多数要求"物理 + 化学"
- 浙江特殊：技术（含通用技术与信息技术）是一门选考科目

## 四、关键变量：赋分制

新高考再选科目（含 3+3 的全部 3 门）采用**等级赋分**：
- 卷面分按全省排名转换为等级，再按等级映射到 100 分制
- 影响：再选科目"考多少分"≠"卷面成绩"，志愿填报时要看赋分后的总分

## 五、志愿填报单位的差异

新高考省份的志愿填报单位也不同：

| 单位 | 代表省份 | 操作方式 |
|------|---------|---------|
| 院校专业组 | 湖北、湖南、广东、江苏、福建、河北、辽宁、重庆 | 一所学校多个组合，每组对应不同选科要求 |
| 专业（专业类）+ 院校 | 浙江、山东 | 直接填到具体专业，志愿数量更多（80-96 个） |
| 院校 + 专业 | 上海、北京、天津、海南 | 介于两者之间 |

**对填报策略的影响**：
- "专业 + 院校"模式下，"是否服从调剂"概念消失，每个志愿就是一个具体专业
- "院校专业组"模式下，组内专业之间还可以服从调剂

## 六、检索建议

每年填报前，必须以"YYYY 年 [省份] 高考志愿填报指南"为关键词检索当年最新政策，因为：
- 改革批次在变
- 选科要求每年由各高校重新公布
- 志愿数量与批次设置可能调整

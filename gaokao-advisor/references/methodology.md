# 志愿填报方法论速查

## 红线规则

**位次禁止凭印象估算**。模型对"分数→位次"的映射极不可靠：
- 例：四川 2025 理科 525 分，模型记忆可能给出"约 80000 位次"，实际官方数据为 **9442 位次**，误差 8 倍
- 例：北京 660 分常被估为"约 5000 位次"，实际 2025 仅 **2153 位次**

任何位次推断**必须当场用 联网检索 查当年一分一段表**。绝不允许在报告中使用未经验证的位次估算。

## 一、为什么位次比分数重要

每年高考试题难度不同，同样 600 分在不同年份可能对应完全不同的全省排名。**位次（全省排名）是跨年度可比的唯一稳定指标**。

- 一分一段表：每年各省考试院公布，列出每个分数对应的全省累计人数
- 操作方法：先把学生分数转换为位次，再用位次去对照目标院校近 3 年录取最低位次

## 二、冲稳保梯度的设计逻辑

### 用"位次差比" R_u/R_s 替代百分比偏移

设考生位次为 R_s，目标院校 N-1 年录取最低位次为 R_u。

| 档位 | 判定规则 | 含义 |
|------|---------|------|
| 冲 | R_u 在 R_s × 0.5 到 R_s × 0.9 区间 | 比考生高 10-50% 档次的院校 |
| 稳 | R_u 在 R_s × 0.9 到 R_s × 1.1 区间 | 真正匹配 |
| 保 | R_u 在 R_s × 1.1 到 R_s × 1.5 区间 | 保底 |
| 不可达（删除） | R_u < R_s × 0.5 | 院校录取位次低于考生位次的一半，纯粹够不着 |
| 浪费（删除） | R_u > R_s × 1.5 | 院校录取位次比考生高 50% 以上，保底也不该这么低 |

> 注：R_u、R_s 仅在内部规则中使用，**不出现在用户报告中**。报告中改用"录取门槛比考生位次高/低 X%"。

### 典型误判模式（务必避免）

**误用"百分比偏移"思路**：

考生江苏物理类 615 分 / 22495 位次。若用"录取分比考生分高 10-20% 算冲档"的分数差思路，会把东南大学 06 组（录取位次 2618）当冲档——实际 R_u/R_s = 2618/22495 = **0.116**，考生位次比东南录取低 8.6 倍，**完全不可达**。

R_u/R_s < 0.5 的院校必须直接移除，避免给考生虚假希望。位次比值才是唯一判据，分数差不可作为档位划分依据。

### 标准比例（占比）

| 档位 | 占比 | 心理预期 |
|------|------|---------|
| 冲 | 约 30% | 中签是惊喜，落选是常态 |
| 稳 | 约 50% | 期望命中区间 |
| 保 | 约 20% | 兜底，确保不滑档 |

### 比例调整建议

- **风险偏好高**（学生愿意复读 / 家庭支持）：冲 40% / 稳 40% / 保 20%
- **风险偏好低**（必须一次上岸）：冲 20% / 稳 50% / 保 30%
- **新高考省份首年**：保档比例至少 30%，因为大小年波动可能加剧

## 三、平行志愿的运行机制

平行志愿规则：**分数优先、遵循志愿、一轮投档**。

- 系统按分数从高到低对所有考生排序
- 轮到某考生时，按其填报的 A→B→C→...→N 顺序检索，第一个达到投档线的院校即投档
- **一旦投档，后续志愿作废**；如果被投档院校退档（不服从调剂、单科限制等），直接进入征集志愿
- **核心结论**：必须勾选"服从专业调剂"以最大降低退档风险；志愿之间必须拉开梯度

## 四、新高考选科限制

3+1+2 模式（湖南、湖北、广东、江苏、福建、河北、辽宁、重庆等）：
- 必选 1 门：物理 或 历史（决定首选科类）
- 再选 2 门：化学 / 生物 / 政治 / 地理 中任选 2

3+3 模式（北京、上海、天津、海南、浙江、山东）：
- 6 选 3 自由组合（浙江为 7 选 3）

**选科直接决定可报专业**：例如绝大多数医学、计算机、电子信息类专业要求"物理 + 化学"组合；不选化学会被大量工科专业拒之门外。

## 五、常见决策误区

| 误区 | 纠正 |
|------|------|
| "只看学校档次，专业随便" | 大类招生 / 转专业政策能否兜底？冷门专业可能直接影响就业 |
| "只看分数线，不看位次" | 跨年度不可比，会严重误判 |
| "去经济发达地区肯定好" | 同档次院校在一线录取分数会高出一截，性价比要算 |
| "热门专业 = 好就业" | 计算机、金融历史上都经历过周期；红绿牌每年在变 |
| "服从调剂会被坑" | 不服从调剂的退档代价远大于被调剂到稍冷专业 |
| "保底学校随便填" | 保底也要是真心愿意去的，否则退档/复读两难 |

## 六、特殊招生通道（按需提示）

- **强基计划**：部分顶尖高校的基础学科（数学/物理/化学/生物/历史/哲学/古文字学等）特殊招生，校测成绩占比较大
- **综合评价**：部分省份（江苏、浙江、广东、山东等）开放，高考成绩 + 校测 + 学考综合录取
- **国家专项 / 地方专项 / 高校专项**：面向农村及贫困地区考生
- **中外合作办学**：学费高（一般 6-10 万/年），但同校录取分数更低，适合家庭经济允许且看重学校牌子的考生

# 35 周可执行日历生成器

## 总原则

学生最大的获得感来自**"打开 PDF 就知道明天翻哪一页"**。本模块定义如何把"基础阶段：数学一轮 + 英语单词"这种模糊话，落到 35 周 × 7 天 = 245 个具体动作。

## 模板结构

每周输出 1 张 7 行表，每行包含：

| 字段 | 示例 |
|------|------|
| 日期 | 2026-05-15（周四）|
| 公共课 | 单词：恋词 abandon-block，60 词，30 分钟 |
| 专业课 | 李良荣《新闻学概论》（第 7 版）第 1 章"新闻的基本特点"，P3–P28 |
| 配套视频 | B 站陈雪《新闻学概论》1.1–1.3 节，60 分钟 |
| 输出 | 整理 3 张概念卡片（新闻定义 / 新闻价值 / 新闻真实性） |
| 总时长 | 8h |

## 时间分配规则

### 每日 8h 学习时长分配（在校全职模板）

```
上午 8:30–11:30（3h）：精力高峰 → 最难科目（数学/专业课理论）
下午 14:00–16:30（2.5h）：精力中等 → 英语 / 专业课实务
晚上 19:00–21:30（2.5h）：精力低 → 单词 / 案例 / 笔记整理
```

### 周末安排

- 周六：视为正常学习日，可加入模考
- 周日上午：复盘 + 错题
- 周日下午 + 晚上：**强制休息**（防崩溃）

### 在职模式（每日 4–5h）

```
早 6:00–8:00（2h）：精力高峰 → 最难科目
通勤路上（30min）：单词 APP
晚 21:00–23:00（2h）：英语 / 政治 / 错题
周末每日 8h
```

## 周计划生成的判断逻辑

生成 35 周日历时，按以下决策树推导：

```
1. 备考起点日期 → 实际周数 = ceil((初试日期 - 起点日期) / 7)
2. 阶段切分：
   - 基础：前 8 周（最少）/ 13 周（充裕）
   - 强化：13 周（标准）
   - 冲刺：8 周（标准）
   - 模考：3 周（不可压缩）
3. 教材容量 / 周数 = 每周章节数
4. 单词总量 / 周数 = 每周词量
5. 每日时长 → 每日 3 任务 / 5 任务 比例
```

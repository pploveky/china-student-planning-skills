# 院校调研指南（22 字段决策矩阵）

## 总原则

每所候选院校必须查齐 22 个字段，缺失项明确写"官方未公开"或"未查得（建议 9 月招生简章公布后补全）"，禁用 "-"、"待补"。

每条关键数据必须标注来源 URL + 查询日期。

## 22 字段清单

### A. 基础数据（10 项，从研招网 / 学院官网爬取）

| # | 字段 | 优先来源 | 备注 |
|---|------|---------|------|
| 1 | 学校层次 + 学科评估等级 | 教育部学位中心 cdgdc.edu.cn | 985/211/双一流 + A+/A/A-/B+ 等 |
| 2 | 学院与专业代码 | 研招网专业目录 | 6 位专业代码必须完整 |
| 3 | 学制学费 | 学院官网招生简章 | 学硕一般 8000/年，专硕差异大 |
| 4 | 招生总数（含/不含推免） | 学院官网拟录取名单 | 必须区分 |
| 5 | 推免占比 | 学院官网推免名单 | 推免 > 60% 须标 【高】 |
| 6 | 复试分数线（近 3 年） | 学院官网复试通知 | 总分线 |
| 7 | 单科线 | 学院官网复试通知 | 单科任一不过即出局 |
| 8 | 报录比 | 研招网或学院公布 | 报名 / 录取 |
| 9 | 参考书目 | 学院官网招生简章 | 注意每年 9 月可能变更 |
| 10 | 复试初试比 | 学院官网复试方案 | 50/50 / 60/40 / 70/30 直接影响逆袭难度 |

### B. 决策矩阵字段（12 项）

#### 11. 录取最低分（近 3 年）

**用途**：复试线只是入门，**录取最低分才是真正门槛**。例如某校复试线 350，但实际录取最低分 378，差 28 分。

**来源**：
- 学院官网拟录取公示名单
- 研招网复试名单
- 知乎/小红书学长贴（最后兜底）

**输出格式**：
```
2025 录取最低分：378 / 复试线：350 / 差距：+28 分
2024 录取最低分：370 / 复试线：345 / 差距：+25 分
2023 录取最低分：365 / 复试线：340 / 差距：+25 分
```

#### 12. 录取均分（近 3 年）

**用途**：合理目标分参考。学生定 380 是合理（接近均分）还是冲（高于均分 20 分）一目了然。

**来源**：将拟录取名单所有成绩求平均（学院官网公示）。

#### 13. 录取本科层次分布

**用途**：**二本/三本生最该看的指标**。如某校 2025 录取 30 人，本科 985+211 占 25 人，二本仅 5 人，那这是隐性歧视高校。

**输出格式**：
```
2025 录取 30 人：985: 12 / 211: 13 / 一本: 4 / 二本: 1 / 其他: 0
本科二本及以下占比：3.3%（隐性歧视风险【高】）
```

**来源**：拟录取名单一般会写本科院校。如未公布，标"无法获取，需联系学长学姐询问"。

#### 14. 复试淘汰率

```
复试淘汰率 = (进复试人数 - 录取人数) / 进复试人数 × 100%
```

**用途**：判断逆袭难度。淘汰率 > 50% 的学校，初试压线进复试基本没戏。

#### 15. 推免与统考录取均分差

**用途**：判断统考公平度。推免均分 380、统考均分 395，说明统考被严格压。

**输出格式**：
```
2025 推免录取均分：380（无初试，仅复试）
2025 统考录取均分：395
统考溢价：+15 分
```

#### 16. 历年压分情况

**用途**：专业课主观题给分倾向。

**来源**（以小红书 / 知乎学长贴为主，标"二手数据，仅供参考"）：
- 搜索关键词：「XX 大学 XX 专业 压分」
- 至少看 3 篇贴互相印证
- 避免单一来源

**输出格式**：
```
压分情况：业界普遍反映"压主观题，名词解释 5 分一题给 2–3 分常见"
来源：知乎话题《XX 大学考研真实情况》（3 个学长经验帖共识）
```

#### 17. 专业课大纲变化

**用途**：备考精力分配。

**输出格式**：
```
2024 → 2025 大纲变化：新增"AIGC 与新闻伦理"专题
2025 → 2026 大纲变化：暂未公布（建议 9 月后查询）
```

#### 18. 复试笔试题型

**输出格式**：
```
复试笔试：3 小时，题型：
- 名词解释：5 题 × 6 分
- 简答：3 题 × 10 分
- 论述：2 题 × 30 分
```

#### 19. 复试常见问题摘录（≥ 10 个）

**用途**：让学生有提早准备的方向。

**来源**：知乎专栏《XX 大学复试经验贴》，至少摘录 10 个真实问题。

**输出格式**：
```
1. 你为什么选择新闻传播学？
2. 谈谈你对算法推荐与信息茧房的看法
3. 你看过哪些新闻传播学经典著作？
... （至少 10 个）
```

#### 20. 3–5 位推荐导师 + 方向

**用途**：复试联系导师参考。

**来源**：学院官网导师介绍页，按学生兴趣方向匹配。

**输出格式**：
```
推荐导师 1：张三教授
- 方向：媒介融合与平台经济
- 近 3 年代表论文：xxx
- 是否还在带学生：是

推荐导师 2：李四副教授
...
```

#### 21. 学院近 1 年重大事件

**用途**：识别"地雷院校"。

**来源**：
- 学院官网公告（搜"通知"）
- 知乎话题《XX 大学 XX 学院》近 1 年帖
- 小红书"考研避雷"话题

**典型事件**：
- 教授离职（影响导师选择）
- 学院方向调整
- 改革传闻（如 2024 起 XX 校将 X 专业从自命题改为统考）
- 历年录取被取消的负面事件

#### 22. 学费 / 奖学金 / 住宿

**输出格式**：
```
学费：8000/年（学硕） / 30000/年（专硕）
奖学金：学业奖学金一等 12000，覆盖率 30%；助学金 6000/年（全员）
住宿：全日制提供，6 人间，1500/年
非全日制不提供住宿
```

## 数据来源优先级

1. **第一优先**：目标院校研招网 / 研究生院 / 学院官网（绝对权威）
2. **第二优先**：中国研究生招生信息网（yz.chsi.com.cn）
3. **第三优先**：教育部学位中心（cdgdc.edu.cn）—— 学科评估
4. **第四优先**：知乎专栏 / 小红书话题（仅交叉验证，必须 ≥ 3 来源印证）

```
### XXX 大学 - XX 学院 - XX 专业（专业代码）

**A. 基础数据**

| 字段 | 内容 |
|------|------|
| 学校层次 | 211 / 双一流 / 学科评估 B+ |
| 学制 | 3 年学硕 |

| 单科线（2025） | 政治 47 / 英语 47 / 业务课 71 |
| 报录比（2025） | 7.1:1 |
| 复试初试比 | 60% / 40% |
| 参考书目 | 1. xxx；2. xxx |

**B. 决策矩阵**

| 字段 | 内容 |
|------|------|
| 录取最低分（2025） | 378（差复试线 +28） |
| 录取均分（2025） | 392 |
| 本科层次分布 | 985: 7 / 211: 8 / 一本: 2 / 二本: 1 |
| 二本占比 | 5.6%（隐性歧视【中】） |
| 复试淘汰率（2025） | 40% |
| 推免与统考均分差 | 推免 380 / 统考 392，统考溢价 +12 |
| 压分情况 | 名词解释普遍偏严，3 篇学长贴印证 |
| 大纲变化 | 2024→2025 新增 "AIGC 与新闻伦理" |
| 复试笔试题型 | 名词×5+简答×3+论述×2，3 小时 |
| 复试常见问题 | 见附录（10 题） |
| 推荐导师 | 张三 / 李四 / 王五（详见附录） |
| 近 1 年大事 | 2025-09 学院新增"国际传播"硕士点 |

**风险等级**：【中】

- 风险点 1：录取最低分 378 高于复试线 28 分，**学生若停留在线上即出局**
- 风险点 2：二本占比 5.6%，本科背景影响显著
- 机会点 1：推免占比仅 40%（同档院校通常 50%+）
- 机会点 2：2025-09 新增专业方向，2026 招生有扩招可能

**推荐档位**：稳

**推荐理由**（必须 ≥ 3 条）：
1. 录取均分 392 与学生合理目标 390 仅差 2 分，符合"稳"档定义
2. 二本占比虽低但仍有先例（每年 1–2 人），初试 395+ 可化解
3. 推免占比 40% 低于同档均值 55%，统考席位友好

**数据来源**
- yz.xxx.edu.cn/xxx ，查询日期：2026-XX-XX（一手）
- 知乎《XX 大学复试经验贴》，查询日期：2026-XX-XX（二手）
```

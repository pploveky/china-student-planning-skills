# 独家情报模块

每份报告必须包含 3 个独家章节，且全部带可点击源 URL：

1. 院校近 1 年舆情监测（每校 200–500 字）
2. 避坑机构清单（行业级，带可核实投诉编号）
3. 前辈录取数据（同专业上岸者，含本科层次分布）

## 1. 院校近 1 年舆情监测

### 1.1 必爬来源

| 来源 | 用途 | 搜索关键词模板 |
|------|------|---------------|
| 学院官网"通知公告"栏 | 学院官方动作 | 学院近 1 年所有公告标题 |
| 学院官网"师资队伍" | 教授变动 | N-1 年与 N 年师资页对比 |
| 知乎话题《XX 大学》 | 学生爆料 | "XX 大学 XX 专业 + 报考年份" |
| 知乎话题《考研》 | 学院动态 | "XX 大学 推免" / "XX 学院 改革" |
| 小红书话题 | 学生备考体验 | "XX 大学 上岸" / "XX 大学 压分" |
| 微博 | 院校官号动态 | @XX 大学研招办 |
| 中国研究生招生信息网 | 大纲变化 | 院校招生简章对比 |

### 1.2 输出格式

```
**XX 大学 XX 学院近 1 年舆情**

**官方动作**

- YYYY-MM：学院新增"国际传播"硕士点，预计 N+1 年招生 5 人
- YYYY-MM：学院发布《XX 报告》调整培养方案

**师资变动**

- YYYY-MM：知名教授 X 教授离职去 Y 大学（影响 X 方向报考价值）
- YYYY-MM：新引进青年学者 Z 副教授（X 方向）

**学生反馈摘录（来自知乎/小红书 ≥ 3 篇互证）**

- 反馈 1：N 年复试压分明显（[来源](url)）
- 反馈 2：XX 教授不带跨考生（[来源](url)）
- 反馈 3：复试英语口语难度提升（[来源](url)）

**对学生的建议**

- 若志在 X 方向，X 教授离职后建议改报导师 W
- 若复试压分情况持续，目标分应从 380 提升至 390
```

### 1.3 数据校验

- 每条爆料必须有 ≥ 3 个独立来源互证才能入报告
- 单一来源标注"待核实"
- 绝不引用"小道消息""内部传言"
- 公开渠道未查到的师资变动，必须诚实写"本报告未在公开渠道查到"，**不得编造**

## 2. 避坑机构清单

### 2.1 必爬来源

| 来源 | 用途 |
|------|------|
| 黑猫投诉 hei.315.com.cn | 教育机构投诉记录 |
| 中国消费者协会 | 退费纠纷案 |
| 知乎话题《考研机构》 | 学生评价 |
| 小红书 #考研避雷 | 一手吐槽 |
| B 站考研大 V 评测 | 行业评价 |

### 2.2 输出格式（按学科分类）

```
**XX 学科考研机构清单**

| 机构 | 类型 | 评价 | 风险等级 | 来源 |
|------|------|------|---------|------|
| XX 机构 | 专业课 | 老牌机构，资料体系成熟；近 N 年退费纠纷 X 起 | 【中】 | [黑猫投诉编号](url) + [知乎专栏](url) |
| YY 机构 | 专业课 | 体量大，全程班 X 元；师资常变动 | 【中】 | [来源](url) |

**已塌房 / 不建议机构**

- ABC 考研（YYYY-MM 跑路，剩余学员无法退费）（[来源](url)）

**不建议购买的网课系列**

- "独家押题包"类（无机构愿意为押题率背书的均不靠谱）
- 不知名机构的"高端 vip 班"
```

### 2.3 法律免责

- 信息来源必须可公开验证
- 用客观陈述（"3 年退费纠纷 N 起"）而非主观贬损（"骗子机构"）
- 报告底部声明"基于公开信息整理，机构状况可能已变化，学生付费前请自行核实"
- 公开渠道未查到投诉记录的机构，必须诚实写"本报告未在公开投诉平台查到可核实记录"，并提供查询入口（黑猫投诉、消费保等）让学生自查

## 3. 前辈录取数据

### 3.1 必爬来源

| 来源 | 用途 |
|------|------|
| 学院官网拟录取公示 | 全部录取者初试分（最权威） |
| 知乎"上岸贴" | 各科分数 + 备考路径 |
| 小红书"上岸笔记"标签 | 同上 |
| 学院考研群（如能加入） | 同期备考者数据 |

### 3.2 输出格式

```
**XX 大学 XX 专业 N-1 年录取者数据**（样本说明：基于公开经验贴 X 篇综合估算，单科分细节属推算）

**初试分数分布**

| 指标 | 数值 | 数据强度 |
|------|------|---------|
| 录取最低分 | 约 X | 强 / 弱 |
| 录取均分 | 约 Y | 强 / 弱 |
| 录取最高分 | 约 Z（[来源](url)） | 强 |

**各科平均分**

| 科目 | 均分 | 满分 | 推断 |
|------|------|------|------|
| 政治 | 71 | 100 | 正常 |
| 英语二 | 76 | 100 | 正常 |
| 334 | 112 | 150 | 偏严（业界普遍 115–125） |
| 440 | 130 | 150 | 正常 |

**结论**：334 偏严是该校主要压分点。

**本科层次分布**（基于 N 篇上岸贴自述统计，仅供参考）

- 985：X（X%）
- 211：X（X%）
- 一本：X（X%）
- 二本：X（X%）
- 三本：X（X%）

**结论**：二本上岸先例存在但稀缺，初试需达到 X+。

**录取者备考路径摘要**

- 路径 A（占比约 X%）：自学 + 院校机构资料
- 路径 B（占比约 X%）：报班全程
- 路径 C（占比约 X%）：自学 + 学长 1v1 答疑

**结论**：自学路径占比 X%+，不必报班。
```

### 3.3 数据真实性校验

- 拟录取公示数据必须可追溯到学院官网
- 自述数据（知乎/小红书）至少 ≥ 5 篇方可作为统计样本
- 样本量 < 5 时标注"样本不足，仅供参考"
- 单点样本（如某状元贴）不可直接外推为均值
- 所有估算数字须附"数据强度"列，弱数据必须说明推算路径

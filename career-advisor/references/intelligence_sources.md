## 二、4 大骗局识别

### 骗局 1：培训贷 / 包就业陷阱

**话术**：

- "免费 / 低价 IT / 设计 / 视频剪辑培训，包就业，月薪过 8k"
- "先免费学，结业后用工资分期还学费"
- "实习名额，但需要先交 X 元保证金"

**真相**：

- 学费 1.5–3w，多数挂在网络贷款上（你贷的不是培训费，是网贷）
- "包就业"多为外包公司岗位（薪资极低，转正无望）
- 培训质量差到工作面试都过不了
- 退费极难，多数有"7 天无理由"之外的合同陷阱

**识别**：

- 要求提交身份证 + 银行卡 + 实名认证 → 100% 骗局
- 推荐去某个"合作公司"实习要交保证金 → 100% 骗局
- 包月薪 X k 是合同写明吗？多数只是口头承诺

**正确做法**：

- 培训机构必须看是否在《全国校外培训机构信息服务平台》备案
- 学费一次性付清（拒绝分期 / 网贷）
- 任何"包就业"承诺必须写进合同

### 骗局 2：假外包 / 假内推

**话术**：

- "我是 [大厂] HR，可以内推，但需要交内推费 / 培训费"
- "我们是 [大厂] 合作公司，工资和正式员工一样"

**真相**：

- 大厂正式 HR 不会私下找学生收内推费
- "合作公司"= 外包公司，工资低 30–50%、福利缺失、转正无望
- 项目结束后立即解约

**识别**：

- 工作合同上是 [大厂] 还是 [外包公司]？必须看合同甲方
- 五险一金谁缴？外包公司缴和大厂缴差别极大
- 工号 / 工卡是不是 [大厂] 编号？

**正确做法**：

- 内推免费，任何收费都是骗局
- 入职合同必须看清甲方 / 工资 / 五险一金缴纳单位

### 骗局 3：空头 Offer / 暗坑岗位

**话术**：

- "我们是知名公司，你被录取了"
- "Offer 已发，请尽快回签三方"
- "薪资面议，入职后会有惊喜"

**真相**：

- 部分小公司用"知名"包装，实际是壳公司
- 入职后岗位与 JD 完全不符（说好做产品，实际做销售）
- "面议"是因为薪资远低于市场水平，怕你比较后拒绝

**识别**：

- 公司是否在企查查 / 天眼查能查到注册资本 / 股东 / 历年诉讼？
- Offer 上是否明确写 base + 浮动 + 五险一金缴纳基数 + 试用期？
- HR 是否能提供办公地址 + 实地参观？

**正确做法**：

- 拿到 Offer 必须有书面文件（PDF / 邮件）
- 入职前要求先到办公地参观
- 入职合同上的工资必须 ≥ Offer 上的工资

### 骗局 4：海外劳务 / 境外就业陷阱

**话术**：

- "高薪境外工作，X 国 / 东南亚 / 中东，月薪 3w+，包食宿"
- "免费签证 / 包机票"

**真相**：

- 部分指向缅甸、柬埔寨等地的"电诈园区"
- "境外护照办理代办"是涉嫌偷渡
- 高薪+包食宿+免费签证 = 99% 是骗局

**识别**：

- 任何要求你"先到 [境外地点] 培训"或"必须从泰国 / 越南陆路过去"= 100% 电诈
- "高薪 + 不需要语言"= 100% 不正规
- 公司能在国内查到正规登记吗？海外业务是否有正规跨境运营资质？

**正确做法**：

- 境外就业必须通过国家商务部或外交部备案的劳务公司
- 薪资合同必须经境外目标国合法雇主签字
- 签证类型必须是工作签证（W 签证），不是旅游 / 商务签证

## 三、问题公司清单（如何识别）

> 警告：以下为通用识别方法，不是特定公司"黑名单"。具体公司须基于公开信息核实。

### 问题公司红旗（满足 3 项以上慎入）

1. **企查查 / 天眼查显示**：
   - 近 1 年劳动争议判决 > 5 起
   - 法定代表人是历次成立公司的"老板老板"（连环创业 + 关闭）
   - 注册资本极低（< 100w）但说"知名公司"

2. **看准网 / 脉脉显示**：
   - 整体评分 < 2.5（满分 5）
   - 近 6 个月负面帖数量明显增加
   - 多人提到"承诺工资未兑现""恶意辞退"

3. **裁判文书网显示**：
   - 多次被员工起诉劳动报酬 / 经济补偿金
   - 多次被起诉社保未缴纳

4. **校招行为异常**：
   - 海投 + 秒发 Offer（连面试都没有）
   - 笔试 / 面试时长 < 10 分钟
   - 入职前要求"押金""保证金""服装费""设备费"
   - 工作地点频繁变动 / 注册地址与办公地不符

5. **行业舆情**：
   - 近 1 年财经媒体多次负面报道
   - 母公司 / 关联公司被监管处罚
   - 近 6 个月有过裁员潮

## 五、户档处理黑红榜

### 红榜（推荐做法）

1. **入职互联网 / 民企前**：
   - 户口落公司所在地集体户（如深圳 / 杭州 / 苏州 / 成都，门槛极低）
   - 档案放公司所在地人才市场
   - 派遣证写公司所在地

2. **进央国企 / 选调 / 公考**：
   - 户口 / 档案直接进单位
   - 不要中转任何"人才中心"

3. **二战考研 / 留学**：
   - 户口回户籍地 / 留在原学校（部分学校允许保留 1 年）
   - 档案放户籍地人才市场（不挂靠学校）

4. **不工作 + Gap**：
   - 户口 / 档案回户籍地
   - 不挂靠"街道办"（可能影响后续公考审核）

### 黑榜（避免做法）

1. **挂靠"假公司"骗户口**：违法且后续暴露后会被取消户口
2. **签三方就违约**：违约金 + 学校诚信记录 + 公司报告 + 个人征信影响
3. **档案不知去向**："档案丢失"在事业编 / 公务员入职时是巨大障碍
4. **多份三方一起签**：违法且会被学校发现
5. **隐瞒已签三方再签新 Offer**：背调被发现 = 立即解约 + 进入行业黑名单

### 户籍地选择决策树

```
进互联网 / 民企
  - 一线（北京 / 上海）：能解决户口 = 留；不能解决 = 留户籍地
  - 强二线（杭州 / 苏州 / 深圳 / 广州 / 成都）：直接落户工作所在地
  - 三四线：根据父母 / 伴侣意愿 + 5 年内回流概率决定

进央国企 / 选调
  - 直接进单位户口

进体制内 + 户籍地不同
  - 优先回户籍地省考（信息差 + 父母人脉）
  - 已在工作地落户后回户籍地考公会复杂（须迁回）

不工作 + 二战 / 留学 + Gap
  - 户口 / 档案回户籍地
  - 不要折腾外地挂靠
```

## 六、薪资爆料数据采集流程（标准化）

```
Step 1：去 OfferShow [https://www.offershow.cn/] 搜公司名 + 岗位
  → 抓 P25 / P50 / P75 三档

Step 2：去 一亩三分地 校招板块搜公司名
  → 交叉验证 P50 是否一致

Step 3：去 看准网薪酬榜
  → 看校招 / 应届工资分布

Step 4：去 脉脉职言搜"X 公司 校招 工资"
  → 看是否有"近 1 年降薪"舆情

Step 5：去 知乎搜"X 公司 应届 工资"
  → 长文细节（如试用期 / 五险一金缴基 / 股票 vest）

最终：
- 三个来源 P50 一致 → 强数据
- 一个来源数据偏差 > 30% → 须标"二手数据，仅供参考"
- 仅一个来源 → 标"待核实"
- 无任何爆料 → 写"未在公开渠道查到，建议通过 HR 直接确认"
```

## 七、应届生应对劳动维权

### 法律基础

- **应届生劳动合同**：受《劳动合同法》保护
- **三方协议**：受民法典合同编保护，违约责任以违约金为限
- **试用期**：最长 6 个月（合同 3 年以上）；试用期工资 ≥ 80% 转正工资 ≥ 当地最低工资
- **加班费**：工作日 1.5 倍，周末 2 倍，法定假日 3 倍
- **社保 / 公积金**：必须缴纳，不缴可投诉

### 维权途径

| 方式 | 适用场景 | 时效 |
|---|---|---|
| 内部沟通 | 一般纠纷 | 1–2 周 |
| 12333 劳动监察热线 | 加班费 / 工资拖欠 / 社保不缴 | 60 天内立案 |
| 劳动争议仲裁 | 大额纠纷 / 解雇争议 | 1 年内提起 |
| 法院诉讼 | 仲裁不服 | 仲裁后 15 天内 |

### 维权前必备证据

- 劳动合同 / 三方协议 PDF
- 工资条 / 银行流水
- 入职 / 离职证明
- 加班记录（钉钉 / 飞书 / 邮件）
- 微信沟通截图（HR / 老板）

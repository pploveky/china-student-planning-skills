# 行业研究指南：8 大行业景气度判断

本文档给出"判断一个行业是否值得应届生进入"的标准化方法 + 8 大热门行业的当前景气度判断。

---

## 行业景气度判断五维框架

判断一个行业是否值得应届生进入，看五个维度：

| 维度 | 判断方法 |
|---|---|
| 1. 营收增速 | 看头部上市公司近 3 年财报营收增速。> 15% 朝阳；5–15% 平稳；< 5% 或负 = 收缩 |
| 2. 利润率 | 看行业平均净利润率。> 15% 高利润；5–15% 普通；< 5% 微利 |
| 3. HC 增减 | 看头部公司近 1 年校招 HC 同比变化。同比增 = 扩张；同比减 ≥ 20% = 收缩 |
| 4. 政策环境 | 国家"十四五"或"十五五"重点支持？还是被点名整改？ |
| 5. 替代风险 | 5 年内是否被 AI / 自动化 / 新技术替代？ |

---

## 行业 1：互联网

### 当前景气度：【中】，从巅峰下行

- **营收增速**：头部公司 2024 年同比 5–15%（已从过去 30%+ 增速明显放缓）
- **利润率**：头部 15–25%（高利润但下降）
- **HC 减**：2023–2025 年校招 HC 较 2021 年峰值减少 30–50%；技术岗收紧但仍是大头
- **政策环境**：反垄断常态化，金融业务（蚂蚁、京东数科）受限
- **替代风险**：AI 已替代基础测试 / 客服 / 部分前端 / 内容审核岗位

### 头部公司 HC 节奏（2025 校招参考）

> 数据来源：各公司官方公告 + 牛客网 / 看准网交叉验证。具体当年数据须现查现核。

- 字节跳动：仍是最大招聘方，国内 + 海外（TikTok）业务双线
- 阿里：调整后聚焦电商 + 云，HC 收缩明显
- 腾讯：金融科技 + 游戏 + 视频号是重点
- 美团：到家业务稳定，本地生活仍在扩张
- 拼多多：低 HC 高强度高薪
- 华为：终端 + 云 + 鸿蒙生态，技术岗大户
- 小米：汽车业务带动新一轮扩张
- B 站、知乎、贝壳：基本停止扩招

### 应届生判断

- 985 / 211 + 大厂实习：互联网仍是高薪首选，但要做好"前 3 年高薪 + 35 岁门槛"心理准备
- 双非 + 无大厂实习：进互联网大厂概率 < 10%，建议同步准备国央企信息技术岗

## 行业 2：新能源（光伏 / 锂电 / 新能源车）

### 当前景气度：【中高】，从狂飙转平稳

- **营收增速**：头部 5–20%（行业从 100%+ 增速转入平稳）
- **利润率**：光伏组件 5–10%（卷价格）；锂电池 10–15%；新能源车 0–10%
- **HC 增**：宁德时代、比亚迪仍在大规模招理工科应届
- **政策环境**：碳中和长期国策；地方政府强力支持
- **替代风险**：低，行业仍是 10 年级别朝阳

### 头部公司

- **锂电**：宁德时代（CATL）/ 比亚迪 / 中创新航 / 国轩高科
- **光伏**：隆基绿能 / 晶澳科技 / 通威股份 / 天合光能
- **车企**：比亚迪 / 蔚来 / 小鹏 / 理想 / 长安 / 吉利 / 上汽 / 一汽
- **整车零部件**：宁德时代 / 华为车 BU / 拓普集团 / 福耀玻璃

### 应届生判断

- 机械 / 电气 / 材料 / 化工 / 自动化 / 控制本硕：大量需求，常驻工厂（江苏溧阳、福建宁德、广东深圳、四川成都、合肥）
- 接受非一线常驻 + 重资产周期：高薪起跳（机械 / 电气硕士 18–28w）

---

## 行业 3：半导体 / 集成电路

### 当前景气度：【高】，国家重点扶持

- **营收增速**：头部 10–25%
- **利润率**：设计公司 15–30%；制造公司 5–15%
- **HC 增**：中芯国际、长电科技、华虹宏力、上海华力等大幅扩招
- **政策环境**：国家级"科技自立"重点
- **替代风险**：低

### 头部公司

- **设计**：海思（华为）/ 紫光展锐 / 韦尔股份 / 兆易创新 / 寒武纪 / 龙芯
- **制造**：中芯国际 / 华虹半导体 / 长江存储 / 长鑫存储
- **封测**：长电科技 / 通富微电 / 华天科技
- **设备**：北方华创 / 中微公司 / 上海微电子

### 应届生判断

- 微电子 / 集成电路 / 物理 / 电子工程本硕：硕士起 18–35w，博士 30–60w
- 985 / 211 微电子专业首选；其它专业难进设计岗，可冲制造 / 封测

## 行业 4：生物医药

### 当前景气度：【中低】，资本退潮后调整中

- **营收增速**：头部 0–10%
- **利润率**：差异巨大（仿制药 10–20%；CXO 5–15%；创新药多数亏损）
- **HC 减**：CXO（药明康德、康龙化成）2024 年明显收缩；biotech 大量裁员
- **政策环境**：医保集采压价 / 创新药审批加速
- **替代风险**：中（AI 制药正在改变研发模式）

### 头部公司

- **传统药企**：恒瑞医药 / 复星医药 / 石药集团 / 扬子江
- **创新药**：百济神州 / 信达生物 / 君实生物 / 复宏汉霖
- **CXO**：药明康德 / 凯莱英 / 泰格医药 / 康龙化成
- **器械**：迈瑞医疗 / 联影医疗 / 微创医疗

### 应届生判断

- 药学 / 生物 / 化学硕博：硕士 12–20w，博士 25–40w（外企更高）
- 行业寒冬中，建议同步考虑教师编 / 制药外企 / 公务员（药监 / 卫健委）

## 行业 5：金融（银行 / 券商 / 保险 / 基金）

### 当前景气度：【中】，分化严重

- **营收增速**：银行 0–5%；券商 -10–15%（吃天气饭）；保险 0–10%
- **利润率**：银行 25–35%；券商 15–25%；保险 5–10%
- **HC 减**：六大行总行 HC 稳定；分行扩招；券商投行近 2 年大幅收缩
- **政策环境**：金融降薪 / 反腐 / 限薪令明显
- **替代风险**：柜员中等（线上替代）；研究员中（AI 写报告）；交易员高（量化）

### 头部公司

- **六大行**：工建中农交邮（先报总行管培，分行起步柜员）
- **股份行**：招商 / 平安 / 兴业 / 浦发 / 中信 / 民生
- **券商**：中信 / 中金 / 海通 / 华泰 / 国君
- **保险**：中国人寿 / 平安 / 太保 / 新华

### 应届生判断

- 金融 / 经济 / 数学 / 财务硕士：银行总行 18–30w，分行 12–20w；券商投行 25–60w（但 HC 极少）
- 银行降薪明显，长期看更稳定；券商高薪但内卷严重

---

## 行业 6：制造 / 汽车 / 工程

### 当前景气度：【中】，传统燃油车收缩 / 智能化转型

- **营收增速**：传统主机厂 -5–5%；新能源车 10–30%；机械 0–10%
- **HC 减**：传统燃油车（一汽 / 上汽 / 广汽 / 北汽）裁员明显；新能源车扩招
- **政策环境**：智能制造 / 高端装备国家战略
- **替代风险**：低（高端制造）/ 高（低端组装）

### 头部公司

- **传统主机厂**：上汽 / 一汽 / 东风 / 广汽 / 北汽 / 长城
- **新能源车**：比亚迪 / 蔚来 / 小鹏 / 理想 / 零跑 / 哪吒
- **零部件**：宁德时代 / 福耀玻璃 / 拓普集团 / 华阳集团 / 均胜电子
- **工程机械**：三一 / 徐工 / 中联 / 柳工 / 山推

### 应届生判断

- 机械 / 车辆 / 自动化 / 电气本硕：新能源车主推（智能驾驶 25–45w）
- 接受三四线工厂常驻 + 出差海外（中东、东南亚、欧洲）

## 行业 7：教育 / 培训

### 当前景气度：【中低】，K12 转型后探底

- **营收增速**：双减后 K12 大裁员；新东方 / 好未来转向素质 + 出海；考研培训仍坚挺
- **利润率**：好未来 5–10%；新东方 10–15%（已转型）
- **HC**：考研培训（中公、华图、粉笔）相对稳定；K12 公办教师编是新出口
- **政策环境**：双减政策延续 / 鼓励职业教育
- **替代风险**：高（AI 个性化辅导冲击补习模式）

### 头部公司

- **培训机构**：新东方 / 好未来 / 中公 / 华图 / 粉笔 / 高途
- **公办教师**：各市教育局教师编（4–8 月集中招聘）
- **职业教育**：中职 / 高职扩招

### 应届生判断

- 师范类应届：首选教师编（深圳 / 上海 / 杭州一线门槛 985 硕）
- 跨专业应届：考教资 + 教师编是体制内第二条路
- 不建议进入 K12 培训机构本部岗位

---

## 行业 8：政府事业（除公务员外）

### 当前景气度：【高】（稳定，非"扩招"概念）

- **HC**：相对稳定，编制改革下事业编缩减但合同制扩张
- **政策**：参公事业单位（如证监会、银保监会派出机构、海事局、海关辅助岗）改革进行时

### 头部岗位

- 央国企总部党群 / 行政
- 高校行政（行政编多数转合同制）
- 海关 / 移民 / 边检 / 海事
- 公益事业单位（博物馆、图书馆、科技馆）

### 应届生判断

- 党员 + 学生干部 + 985 硕：选调 + 央企党群 + 公考机关三线并行
- 普通双非：地市事业编 + 教师编 + 三支一扶

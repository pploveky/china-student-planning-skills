# 简历与面试材料：8 模块 + STAR 30 题 + 反向提问

应届生最常见的"出工不出活"是简历投了 200 家但没人面，或者面试机会拿到了却答得一塌糊涂。本文档给出"过 80% HR 筛"的简历结构 + 90% 大厂会问的 STAR 30 题 + 让面试官记住你的反向提问。

---

## 一、简历必含 8 模块（按顺序）

### 模块 1：基础信息（4–5 行内）

```
姓名 | 性别（部分岗位有偏好可选）| 求职意向（一行写清）
学校 + 学历 + 专业 + 毕业年份
所在地 / 期望地 + 联系电话 + 邮箱
政治面貌（党员 / 预备党员，体制内必写；外企可不写）
个人主页 / GitHub / 作品集（如有）
```

避坑：

- 不写身高 / 体重 / 婚姻 / 籍贯（除非央企岗位 JD 要求）
- 不放与求职无关的微信 / QQ
- 邮箱用 Gmail / 163，**不要用 QQ 邮箱**（外企 / 头部公司多有偏见）

### 模块 2：教育背景

```
本科 / 硕士 院校全名 + 学院 + 专业 + 学位
GPA：X.XX / 4.00（专业排名 X / Y，前 X%）
主修课程（与岗位强相关 4–6 门）
奖学金 / 荣誉（国奖 / 校奖 / 优秀毕业生）
```

避坑：

- GPA 低于 3.0 的不写绝对值，只写"专业排名前 X%"
- 主修课程不要罗列 20 门，只挑岗位强相关
- 学校排名不在前 100 的，省略学校"全国排名"宣传

### 模块 3：实习 / 工作经历（按时间倒序）

每段实习严格按 STAR 结构 3–5 条：

```
公司名 + 部门 + 岗位 + 时间段
- [Situation 背景] 在 X 项目中负责 Y 模块
- [Task 任务] 目标是 [量化目标，如 DAU 提升 10%]
- [Action 动作] 通过 [具体方法，如 A/B test + 数据分析]
- [Result 结果] 实现 [量化结果，如 DAU +12%、收入 +$5w]
```

避坑：

- 每条不超 3 行
- 必须有量化数字（用户数 / 收入 / 时长 / 成本节省）
- 不写"参与了""协助了""了解了"（被动词），全部改为"负责""主导""完成"
- 不写公司机密（会被背调出来）

### 模块 4：项目 / 论文 / 竞赛

```
项目名 + 时间 + 角色（队长 / 核心成员 / 个人独立）
- 项目背景与目标
- 技术栈 / 方法
- 个人贡献（不要"我们团队 …"）
- 成果（论文发表 / 比赛奖项 / 上线效果）
```

避坑：

- 大创 / 互联网+ 校级奖项慎写（HR 看了无感）；省级 / 国家级才有用
- 论文一作 / 二作分别标注（多数 HR 不知道作者顺序的含金量，但二作以后基本不加分）
- 学科竞赛只写国家级 / 省级以上（数模国一国二、ACM 区域铜以上、Kaggle 铜以上）

### 模块 5：校园经历

```
学生干部 / 社团：职务 + 时间 + 主要工作 + 量化产出（活动人数 / 经费规模）
志愿服务：累计时长（≥ 100 小时才有意义）
党团活动：发展党员、思政骨干、入党介绍人等
```

避坑：

- 体制内 / 国央企岗位写学生干部 + 党员，互联网 / 外企可省
- 不要"团委干事"+"学院学生会副主席"+"班级团支书"全堆，挑 1–2 个最重的

### 模块 6：技能证书

```
外语：CET-4/6 + 雅思 / 托福 + 商务英语口语自评
计算机：编程语言 + 框架 + 工具（按熟练度排序）
行业证书：CFA / CPA / 法考 / 教资 / 普通话 / 一建 / 注会
```

避坑：

- 编程语言只写熟练以上，不要"了解 Python C++ Java Go Rust"全堆
- 雅思 / 托福低于阈值（雅思 6.0 / 托福 80）不写绝对值
- 教资 / 普通话只在体制内 / 教师岗写

### 模块 7：自我评价（3 行内，可选）

按目标岗位 JD 关键词反推 3 个核心词，不要"沟通能力强 / 团队精神 / 抗压"通用废话。

```
推荐结构：[岗位最重要的 1 个特质 + 例证] + [岗位最重要的第 2 个特质 + 例证] + [职业目标]
```

例：

```
3 段大厂产品实习，从 0 到 1 上线 2 款日活 10w+ 产品；擅长用数据驱动决策（实习期间通过 A/B test 优化新人留存 +18%）；目标在头部互联网做用户增长，3 年内成为业务负责人。
```

### 模块 8：附加信息

- 个人博客 / GitHub / 作品集 URL（必须可访问）
- 出版物 / 公开演讲
- 兴趣特长（与岗位有关才写：摄影 / 街舞除非投设计 / 创意岗位，否则不写）

## 二、行为面试 STAR 30 题库

90% 大厂面试都会问其中 5–10 题。建议每题准备 1–2 段 STAR 故事，5 分钟内能讲完。

### 自我认知（5 题）

1. 请做一个 3 分钟自我介绍，重点突出最匹配本岗位的经历
2. 你最大的优势 / 最大的劣势是什么？分别有具体例子吗？
3. 5 年后的职业规划是什么？
4. 你为什么选择我们公司 / 我们这个岗位？
5. 介绍一段你最有成就感的经历

### 团队协作（5 题）

6. 描述一次你与团队成员意见不合，如何解决
7. 团队中最难合作的一次经历，你做了什么？
8. 在团队中你通常扮演什么角色？为什么？
9. 一次你主动帮助他人完成任务的经历
10. 一次需要跨部门 / 跨学院合作的经历

### 抗压与挫折（5 题）

11. 描述一次你失败的经历，如何复盘？
12. 时间紧 + 任务多时如何应对？
13. 一次受到不公平评价 / 误解时你如何处理？
14. 一次大的目标失败，你如何调整？
15. 一次必须熬夜 / 高强度赶工的经历

### 主动性与领导力（5 题）

16. 一次你提出新想法并推动落地的经历
17. 一次你担任领导者 / 队长的经历，遇到什么挑战？
18. 一次你在没有明确指令的情况下主动行动的经历
19. 一次你说服上级 / 老师 / 队员接受你方案的经历
20. 一次你觉得已有方案有问题，如何 challenge 的经历

### 学习与成长（5 题）

21. 一次你学会一项新技能 / 新知识的经历
22. 你最近在读什么书？最大启发是什么？
23. 一次你犯了错并从中学习的经历
24. 你认为大学最大的收获是什么？
25. 一次你接到完全陌生领域任务时如何应对的经历

### 岗位 / 行业认知（5 题）

26. 你对 [本行业] 当前发展趋势的看法
27. 你认为 [本岗位] 最核心的 3 个能力是什么？为什么？
28. 你最欣赏的 [本行业] 公司是哪家？为什么？
29. 你最近关注的 [本行业] 新闻 / 事件是什么？
30. 用 1 个产品 / 项目展示你对 [本岗位] 的理解

## 三、反向提问 10 题（让面试官记住你）

面试结尾"你有什么问题问我吗？"是隐藏环节，问得好可以扭转整场印象。

### 关于团队 / 业务（推荐每场必问 1 个）

1. 这个团队近 1 年最大的成就是什么？踩过哪些坑？
2. 这个岗位 3 年内的成长路径是什么？前辈做到 P6 / 主管平均花了多久？
3. 团队目前面临的最大挑战是什么？
4. 这个岗位最看重应届生的哪些能力？

### 关于公司 / 行业（展示行业理解）

5. 公司未来 3 年最重要的战略方向是什么？
6. [行业新事件] 对公司业务的影响您怎么看？
7. 公司是否有内部创业 / 内部转岗机制？

### 关于面试官（建立连接）

8. 您加入这家公司的初衷是什么？后来发生了什么变化？
9. 您觉得在这工作最让您骄傲的是什么？

### 关于流程 / 后续

10. 接下来流程是什么？大约什么时候出结果？

避坑：

- 不要问薪资 / 加班 / 节假日 / WLB（HR 面再问）
- 不要问明显能 Google 到的（如"公司有几个事业部"）
- 不要问让面试官为难的（如"团队有过裁员吗"）
- 准备 3 个问题，根据面试官时间挑 1–2 个问

## 四、不同面试场景应对

### 4.1 群面（无领导小组讨论）

- 角色分配：Timer / Leader / Reporter / 普通参与者
- 推荐角色：Reporter（总结发言）或 普通参与者发挥 2–3 次高质量观点
- 避坑：抢 Leader 没意义，被秒掉的多；做 Timer 只是没存在感
- 黄金句式："我同意刚才 X 同学的观点 …，同时补充 …"+"基于刚才大家讨论，建议下一步 …"

### 4.2 技术面（互联网开发岗）

- 必背：数据结构（数组 / 哈希 / 链表 / 树 / 图）+ 算法基础（排序 / 二分 / 动态规划 / 双指针 / DFS BFS）
- 项目深挖：每个项目都要能回答"为什么这样做""换一个方案如何""线上有 bug 怎么排查"
- 必备工具：LeetCode 刷至少 200 题（Hot100 + 公司高频）

### 4.3 群面 / Case 面（咨询 / 投行）

- 框架：Profitability / Market Sizing / M&A / Strategy 四大类
- 资源：Case in Point / Victor Cheng / WSO Case Practice
- 准备：每天 1–2 个 case 模拟，至少 3 个月

### 4.4 公考结构化面试

- 题型：综合分析 / 计划组织 / 应急应变 / 人际关系 / 自我认知
- 框架：先表态 + 分析原因 + 提出对策 + 总结
- 资源：粉笔结构化面试 / 中公面试题集

## 六、应届生最常见简历错误

1. **流水账实习**："参与了 X 项目，协助了 Y 工作"——全无量化，HR 跳过
2. **模板化自我评价**："沟通能力强 / 团队精神 / 抗压能力强"——HR 已 PTSD，全部跳过
3. **过度堆砌**：3 页 PDF，HR 平均看 30 秒看不完
4. **格式问题**：左右页边距不统一 / 字体 5 种 / 颜色超 2 种 / 字号超 4 种 → 设计岗直接淘汰
5. **GPA 失真**：写 3.5 / 4.0 但实际 80 / 100 = 3.0 → 学信网认证后会被发现
6. **照片**：白底证件照 + 西装领带 / 西装套装；不要美颜过度
7. **PDF 文件名**：`姓名+学校+专业+岗位.pdf`（不要"final-final-v3.pdf"）
8. **简历不更新**：投递日期超 3 个月须更新最新实习与项目

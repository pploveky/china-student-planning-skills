# 数据源标准与引用规则

就业报告最容易失信的地方是薪资、裁员、加班、户口、招聘截止日期。所有关键判断必须标注来源强度。

## 1. 来源强度

| 强度 | 定义 | 可用于 |
|---|---|---|
| 强 | 官方招聘页、公司年报、政府公告、学校就业网、法院 / 监管文书 | 招聘时间、岗位要求、财报、落户、三方、政策 |
| 中 | 应届生、牛客、OfferShow、看准、一亩三分地等二手平台，且 2 个以上来源一致 | 薪资区间、面试流程、加班强度初判 |
| 弱 | 单一 UGC 帖、小红书、知乎、脉脉单帖、群聊截图 | 只能提示风险，不能当结论 |
| 未查得 | 公开渠道没有找到 | 必须写未查得，不得编造 |

## 2. 公司字段来源

| 字段 | 优先来源 |
|---|---|
| 招聘入口 | 公司校招官网 / 官方公众号 / 学校就业网 |
| 投递截止 | 公司校招官网 / 宣讲会公告 |
| 岗位 JD | 官方招聘页 |
| 营收利润 | 年报、招股书、巨潮资讯、港交所、SEC |
| 薪资 | OfferShow、看准、一亩三分地、应届生、牛客，至少 2 个来源交叉 |
| 加班 | 看准、脉脉、知乎、牛客面经，必须标二手 |
| 裁员 / 监管 | 财新、第一财经、36 氪、公司公告、监管公告 |
| 户口 / 档案 | 当地人社局、学校就业指导中心、公司 HR 书面答复 |

## 3. 引用写法

正文写法：

```markdown
- 招聘入口：[公司 2026 届校园招聘](https://example.com/campus)
- 薪资：P50 约 25w（二手数据，仅供参考；来源：[OfferShow](https://example.com)、[看准网](https://example.com)）
- 风险：公开渠道未查得近 12 个月大规模裁员报道。
```

禁止写法：

- "多家媒体报道"
- "网上都说"
- "知乎 3 篇互证"
- "具体链接自行查证"
- "详见官网"
- "学长学姐反馈"

## 4. 数据缺失处理

- 招聘截止未查得：写"公开渠道未查得，建议直接向 HR 确认"。
- 薪资未查得：写"公开薪资数据不足，不纳入薪资排序"。
- 舆情未查得：写"公开渠道未查得重大负面舆情"，不能反推"低风险"。
- 户口未查得：写"须以 HR 书面答复为准"。

## 5. CSV 数据强度字段

目标公司 CSV 建议增加：

- `招聘来源强度`
- `薪资来源强度`
- `舆情来源强度`
- `待核实事项`

强 / 中 / 弱 / 未查得必须显式写出。

# 中国学生规划 Skills

这是一组面向中国学生和家长的 Claude / AI Agent Skills，覆盖三个常见决策场景：**高考志愿、考研择校与备考、大学生就业规划**。

这些 Skills 的目标不是替学生做决定，而是把分散的信息整理成可核验、可执行、能复盘的规划材料。学生和家长可以免费自用；如果经济条件允许且觉得有帮助，可以自愿支持一点。如果暂时不方便支持，直接使用也没关系，优先把自己的规划做好。

## 包含的 Skills

### `gaokao-advisor`：高考志愿规划建议师

适用于高考生和家长进行志愿填报前的信息整理与冲稳保方案设计。

核心能力：

- 根据省份、科类或新高考选科、分数、全省位次、兴趣、地域、学费和体检限制生成志愿建议。
- 强制使用可核验来源，不凭模型记忆编造录取分、位次、招生计划或专业组信息。
- 输出冲稳保策略、志愿表填写顺序卡片、退档风险自查、征集志愿应急说明。
- 支持单学生咨询，也支持通过 Excel 模板批量生成任务包。

适合问题：

- “山东物化生 620 分、位次 1.8 万，怎么填志愿？”
- “想去江浙沪学计算机，冲稳保怎么排？”
- “孩子色弱，哪些专业不能报？”
- “河南新高考院校专业组怎么填顺序？”

### `kaoyan-advisor`：考研规划建议师

适用于准备考研的学生进行择校、目标分拆解和全年复习规划。

核心能力：

- 根据本科背景、专业、英语数学基础、跨考情况、目标地域、可投入时间和预算生成个性化初试方案。
- 对候选院校做 22 字段调研，包括分数线、报录比、招生人数、参考书目、压分情况和公开舆情。
- 生成明确的“建议什么 / 不建议什么”结论，而不是泛泛而谈。
- 输出 33–35 周逐日备考日历、资料清单、风险红线和公开信息核验清单。

适合问题：

- “二本计算机想考 211 计算机，目标怎么定？”
- “跨考新传，哪些学校更稳？”
- “还有 8 个月，每周 35 小时，怎么安排英语数学专业课？”
- “一战失败 310，二战还要不要冲原学校？”

### `career-advisor`：就业规划建议师

适用于本科生、硕士、海归和毕业 1–3 年内重新求职者做就业路径选择。

核心能力：

- 在体制内、企业、深造三大路径之间做 A / B / C 分档决策。
- 结合学历、专业、GPA、实习、技能证书、城市偏好、家庭现金流和风险偏好给出就业建议。
- 输出主报告、35 周求职日历和目标公司清单。
- 覆盖秋招春招、考公考编、央国企、互联网、外企管培、跨境就业、Offer 比较、三方协议和户档落户等场景。

适合问题：

- “普通一本机械本科，想留西安，秋招怎么规划？”
- “海归商科硕回国，外企管培和互联网运营怎么选？”
- “家里现金流紧张，是考公、找工作还是二战？”
- “两个 Offer 怎么比较，三方要不要签？”

## 使用方式

把本仓库中的三个目录放到支持 Skills 的客户端目录中，然后在对话里提出相应任务即可。不同客户端的安装路径可能不同，请以你使用的客户端文档为准。

目录结构：

```text
gaokao-advisor/
kaoyan-advisor/
career-advisor/
```

如果需要生成 PDF，通常还需要本地安装：

- `pandoc`
- `xelatex` 或完整 TeX 发行版
- Python 3.10+
- 相关脚本依赖，例如 `openpyxl`、`PyMuPDF`

高考志愿 Skill 内置了一些辅助脚本，例如：

```bash
python3 scripts/parse_students_xlsx.py students.xlsx
bash scripts/md_to_pdf.sh report.md report.pdf
```

就业 Skill 的交付物校验示例：

```bash
python3 scripts/validate_outputs.py \
  --report report.pdf \
  --calendar calendar.pdf \
  --csv companies.csv \
  --student-name 张同学
```

联系时请尽量说明：

- **使用场景**：高考志愿、考研规划或就业规划。

## 重要原则

- **不编造数据**：录取分、位次、招生计划、报录比、薪资、岗位和政策都必须来自可核验来源。
- **不做保过承诺**：这些 Skills 只能辅助分析，不能保证录取、上岸或拿 Offer。
- **不替代官方信息**：高考以省考试院和高校招办为准；考研以研招网、学校研究生院和学院官网为准；就业以公司官网、招聘公告和政府政策为准。
- **不帮助违规行为**：不支持学历造假、简历造假、论文代写、考试作弊、面试代考、伪造证明等请求。
- **保护个人信息**：公开示例、Issue 和截图里请打码姓名、身份证号、准考证号、手机号、邮箱、家庭住址等敏感信息。
